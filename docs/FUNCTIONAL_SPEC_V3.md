
### 3.6 멀티턴 채팅 컨텍스트 (절대 준수)

- 백엔드 `/api/insight`에 `priorTurns` 6턴 전달
- **동일 채팅 세션 안 모든 후속 질문 = 무조건 후속 질문 처리** (CEO 2026-06-16 11:55)
- 이전 대화의 국가/세그먼트/주제를 현재 질문 기본 맥락으로 이어받음
- 명시적 전환 표현("그러면 일본은?", "40대는?")만 새 맥락

### 3.7 사이드바 점프 (답변 → 표준 분석)

- "라이프스타일 표준 분석 →" 같은 Related Insight 버튼 클릭 → `chatBoardJumpToTab('life')`
- 백엔드 `/api/ask`가 첫 질문을 country + filters로 파싱
- mode-chat 종료, mode-insight 진입, 빌더 자동 세팅, `runAnalysis()` 자동 실행

## 4. 타겟 인사이트 모드 (mode-insight) — 페이지 상세

### 4.1 페이지 목적

빌더로 타겟을 명시 선택(국가 + 6대 차원) → 7개 표준 분석 탭에서 합성 패널 데이터 시각화. 채팅의 자연어 진입과 달리 정밀 분석.

### 4.2 화면 영역 구성

```
┌─────────┬───────────────────────────────────────┐
│         │  [Top bar: ← 뒤로 | ? 사용법 | 새 분석 ]│
│         ├───────────────────────────────────────┤
│         │  [Tab 줄: 요약/Who/Life/Mind/...]      │
│ 사이드바 ├───────────────────────────────────────┤
│         │                                       │
│ (240px) │           [선택된 탭 콘텐츠]            │
│         │                                       │
│         │   (빌더는 'overview' 탭에서 노출)       │
└─────────┴───────────────────────────────────────┘
```

### 4.3 Top bar 버튼

| 버튼 | 함수 | 의미 |
|------|------|------|
| **← 뒤로** | `history.back()` | 이전 페이지/모드 복귀 |
| **? 사용법** | `showHelp()` | 사용법 모달 (빌더 사용법, 차원 설명 등) |
| **보고서 PDF** | `downloadReportPDF()` | 현재 분석 결과 전체 PDF (`#tb-pdf-report`) |
| **새 프로젝트** | `saveTarget()` | 현재 빌더 + 분석을 프로젝트로 저장 (`#tb-new-project`) |

### 4.4 탭 줄 (`.ws-tabs`)

| 탭 ID | 라벨 | 함수 | 목적 |
|-------|------|------|------|
| `summary` | 📋 요약 | `switchTab('summary')` | KPI + 핵심 인사이트 3건 |

| `life` | 🌿 라이프스타일 | `switchTab('life')` | 일상·운동·식습관·여행 |
| `mind` | 💭 가치관 | `switchTab('mind')` | 핵심 가치·성격·환경의식 |
| `love` | ❤️ 관심사 | `switchTab('love')` | 관심사 Top10·콘텐츠·음악 |
| `buy` | 🛒 구매 | `switchTab('buy')` | 결정요인·쇼핑채널·지출 |

| 버튼 | 함수 | 의미 |
|------|------|------|
| **필터 빌더로 이동** | `switchTab('overview')` | 차원 선택 화면(빌더)로 |

### 4.5 빌더 (overview 탭) — 세부

#### 4.5.1 국가 패널

| 요소 | 의미 |
|------|------|
| 기준국 (단일 선택, 절대 정책) | `currentCountry`. 77개국 카드 그리드. 국기 + 국가명 + 인구 |
| 비교국 (다중, 최대 3개) | `compareCountries`. compare 탭 매트릭스 표시 시에만 활용 |
| 검색 인풋 | 국가명 부분 일치 검색 |
| 권역 필터 칩 | 아시아/유럽/북미/중남미/중동/오세아니아/아프리카 |

| 그룹 | 색상 | 차원 (예시) |
|------|------|------------|
| 👤 **Who · 인구통계** | 인디고 | 연령·성별·가구소득·교육·가구구성·혼인·자녀수·자녀연령·직업·도시규모·거주형태·민족문화 |

#### 4.5.3 빌더 액션 버튼

| 버튼 | 함수 | 의미 |
|------|------|------|
| **분석 시작** | `runAnalysis()` | 백엔드 `/api/audience/synthesize` 호출 → 합성 패널 N=30 생성 → 모든 탭 데이터 채움 |
| **필터 초기화** | `resetFilters()` | 차원 선택 전부 ['전체']로 |

### 4.6 표준 분석 탭별 상세

#### 4.6.1 요약 (summary)

| 영역 | 내용 |
|------|------|
| KPI 카드 (6개) | 평균 연령, 핵심 가치 Top1, OTT Top1, 관심사 Top1, 구매결정요인 Top1, 환경의식 점수 |
| 핵심 인사이트 (3건) | LLM 생성 한 줄 통찰 + 근거 |
| 페르소나 미리보기 (3개) | 가상 이름 + tagline + 클릭 시 인터뷰 탭 활성 |

#### 4.6.2 인물상 (who)

- 연령 분포 막대 차트 (10대~60대+)
- 성별 도넛
- 가구 소득 10분위 막대
- 직업 Top 10 가로 막대
- 거주 도시 규모 4분위 도넛

#### 4.6.3 라이프스타일 (life)

- 일상 루틴 24시간 타임라인 (수면/일/여가/식사/운동)
- 운동·웰니스 빈도
- 식습관 (외식 vs 자취)
- 여행 빈도·유형
- 사회활동 카테고리

#### 4.6.4 가치관 (mind)

- 핵심 가치관 Top 5 막대
- 성격 유형 (MBTI 16종) 도넛
- 환경 의식·신기술 수용도 게이지
- 삶의 만족도 1~10 분포

#### 4.6.5 관심사 (love)

- 관심사 Top 10 가로 막대 (다중)
- 선호 콘텐츠 장르 Pie
- 음악 장르 + K-콘텐츠 관여도

#### 4.6.6 구매 (buy)

- 결정요인 Top 5 (가격/품질/브랜드/리뷰/지인추천)
- 쇼핑 채널 점유율 (온라인/오프라인/모바일/소셜커머스)
- 카테고리 지출 비중 (의식주·여가·교육·의료 등)
- 프리미엄 vs 가성비 게이지

#### 4.6.7 미디어 (media)

**3가지 뷰 전환 버튼:**

| 버튼 | 함수 | 의미 |
|------|------|------|
| 채널 트리 (Statista 9) | `switchMediaView('tree')` | Statista 9대 미디어 채널 계층 트리 |
| 전체 매체 표 | `switchMediaView('flat')` | OTT/SNS/메신저/동영상/뉴스 전체 표 |
| 차트 | `switchMediaView('chart')` | 점유율 차트 |
| 광고비 요약 | `switchMediaView('spend')` | 광고비 추정 요약 (KOBACO/MEZZO 등) |

#### 4.6.8 페르소나 인터뷰 (interview)

- 페르소나 카드 클릭 → 활성화
- 인터뷰 UI: AI 챗봇 형식 1:1 대화
- 백엔드 `/api/interview` (Gemini)로 페르소나가 1인칭으로 답변
- **호칭 일관성 절대 준수**: 남성 페르소나가 보이그룹을 "오빠"라 부르지 않음 (fix 30)

### 4.7 글로벌 비교 (compare 탭)

- 빌더에서 비교국 선택 (최대 3개) 시 활성
- 기준국 + 비교국 매트릭스
- 5개 차원(인구통계·미디어·관심사·가치관·구매)별 비교 막대 차트
- 데이터 확보 안 된 국가는 회색 처리 + tooltip 안내

6. **인터뷰 질문 2~7**: 주말 일과, 광고 노출 채널, 최근 구매 실패 경험, 친환경 구매 고려 수준 등

## 8. 백엔드 API 라우트 상세

### 8.1 라우트 목록

| 경로 | 메서드 | 핵심 동작 |
|------|--------|----------|

| `/api/catalog` | GET | 차원·옵션 카탈로그 |

| `/api/dimension-insights` | POST | 단일 차원 심층 분석 |
| `/api/dimensions` | GET | 차원 그룹·메타데이터 |
| `/api/insight` | POST | **AI 채팅 답변** (priorTurns 멀티턴) |
| `/api/insights` | POST | 7개 표준 분석 탭 데이터 |
| `/api/interview` | POST | 페르소나 인터뷰 1:1 대화 |

| `/api/research` | POST | 실시간 리서치 (Google Search grounding) |
| `/api/stratify/country` | GET | World Bank 매크로 지표 |
| `/api/trends` | POST | Google Trends 24h 갱신 |

### 8.2 `/api/insight` 요청·응답 스펙

**요청 본문**:
```json
{
  "question": "Z세대 한류 팬의 인스타그램 광고 톤은?",
  "country": "KR",
  "filters": { "age":["20대"], "musicGenre":["K-Pop"] },
  "priorTurns": [
    { "q":"...", "a":{ "headline":"..." } }
  ],
  "synthesized": { "who":{}, "life":{}, ... }
}
```

**응답 본문**:
```json
{
  "inScope": true,
  "headline": "팬덤 언어 + K-콘텐츠 트렌드 용어 + 단 신조어는 풀어쓰기",
  "narrative": "...",
  "kpis": [{ "label":"카카오톡 점유율","value":"92%" }, ...],
  "charts": [{ "type":"bar","data":{...} }],
  "personaSamples": [{ "name":"주민","tagline":"...","attributes":[],"voice":"..." }],
  "sources": [{ "label":"INNOCEAN AI 페르소나 패널 N=30","url":null }, ...],
  "relatedInsights": [{ "tab":"life","label":"라이프스타일 표준 분석 →","reason":"일상 루틴" }]
}
```

### 8.3 신조어·외래어 표기 규칙 (fix 39, 시스템 프롬프트)

```
형식: 한국어 표기(원어 표기, 발음): 의미 한 줄 설명
예:
- 고생 취소(苦労キャンセル, 쿠로 캔슬): 굳이 고생 안 하고 쉽게 가자
- 평타족(平躺族, 핑탕쭈): 노력하지 않고 평평하게 누워 산다는 중국 청년 문화
- 퀵 커머스(Quick Commerce): 30분~1시간 즉시 배송 e-commerce
적용: headline/narrative/voice/tagline/attributes/personaSamples 전체
1회 병기, 동일 답변 내 재등장은 한국어만
```
